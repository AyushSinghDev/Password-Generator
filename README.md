# Password-Generator

**Live demo:** https://password-generator-devxt.vercel.app/  

## Overview  

Password-Generator is a simple web-app that allows users to quickly generate a random secure password. With a single click, it produces a password composed of a mix of characters — helping users create strong passwords for their accounts.

Typical use-cases:  
- When you need a quick, random password for a new account or service.  
- When you want to avoid weak or easily guessable passwords.

## Tech Stack  

- HTML5  
- CSS3
- JavaScript

## Features  

- Generate a random password.  
- Ability to adjust the password generation criteria (e.g. include uppercase, lowercase, numbers, symbols).  
- Simple UI — accessible in the browser, no backend needed.  
- Copy-to-clipboard, so user can easily copy the generated password to use elsewhere.  
- Lightweight and easy to use — no dependencies, works out of the box.
